Initial position, velocity, spin, and toss parameters are adjustable.

## Controls

| Key | Action |
|-----|--------|
| W/S | Move camera forward/backward |
| A/D | Move camera left/right |
| Space/Shift | Move camera up/down |
| Right-click + drag | Rotate camera |
| Play Out | Simulate the ball trajectory |
| Reset | Reset ball to initial state |
